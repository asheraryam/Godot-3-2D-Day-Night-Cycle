# Debug Overlay

## Parameters

### Moon Node Path

| Name | Type | Description |
| - | - | - |
| `moon_node_path` | `NodePath` | If a moon node is assigned, a "Show moon" checkbox will appear in the debug overlay to enable/disable the moon.  |

### Show

| Name | Type | Description |
| - | - | - |
| `show` | `bool` | Enables/disables the debug overlay. |
